# Tech Interview Agent — Golden Eval Dataset

**Stop guessing whether your AI interviewer is good. Start measuring it.**

This dataset provides ground-truth benchmarks for evaluating AI agents that conduct **tech job interviews**. Each record is a structured test case: give it to your agent, collect the response, run it through the [AI Agent Evaluation Pipeline](https://huggingface.co/spaces/build-small-hackathon/AI-agent-Evaluation-pipeline), and get objective scores — no human review needed.

Generated by **NVIDIA Nemotron-3-Nano-30B-A3B**.

---

## What's inside

**40 golden records** across 8 tech interview domains, each containing:

| Field                              | Description                                      |
| ---------------------------------- | ------------------------------------------------ |
| `scenario.user_goal`               | What the interview session should accomplish     |
| `scenario.system_prompt`           | System prompt for the AI interviewer agent       |
| `scenario.initial_message`         | Candidate's opening message to start the session |
| `ground_truth.expected_response`   | What a perfect interviewer would say             |
| `ground_truth.expected_trajectory` | Expected tool calls in order (if any)            |
| `ground_truth.assertions`          | Specific, checkable pass/fail criteria           |

## How to use

```python
from datasets import load_dataset

ds = load_dataset("build-small-hackathon/agent-eval-golden-dataset", split="train")

record = ds[0]

# Feed scenario to your agent
agent_response = my_agent.invoke(
    goal=record["scenario"]["user_goal"],
    system_prompt=record["scenario"]["system_prompt"],
    message=record["scenario"]["initial_message"],
)

# Evaluate against ground truth
from src.runner import EvalRunner
from src.models import GroundTruth

runner = EvalRunner()
report = runner.run(
    session=build_session(record, agent_response),
    ground_truth=GroundTruth(
        expected_response=record["ground_truth"]["expected_response"],
        expected_trajectory=record["ground_truth"]["expected_trajectory"],
        assertions=record["ground_truth"]["assertions"],
    )
)

print(f"Score: {report.overall_score:.0%}")
```
